Create a Blogs section for the Aeronautical Engineering Department website with an aerospace-themed UI (dark navy, matte black, metallic accents, glassmorphism cards, smooth animations).

User Features:

Any logged-in user can write and submit a blog post

Blog submission form fields:

Blog Title

Author Name

Category (Aviation, Space, UAVs, Aerodynamics, Technology, Research)

Cover Image upload

Blog Content (rich text editor)

Tags

Users can edit or delete their own blogs

Submitted blogs should initially have “Pending Approval” status

Admin Features:

Admin dashboard to review submitted blogs

Approve or reject blogs

Edit or delete any blog

Once approved, the blog becomes publicly visible on the website

Blog Page Layout:

Grid or card-based blog listing

Each blog card shows:

Cover image

Title

Author

Date

Read more button

Individual blog page with full content and comments section (optional)

Database Models:

Users

Blogs

Blog Status (Pending / Approved / Rejected)

Categories

Comments (optional)

Extra Features:

Search and filter blogs by category

Like or bookmark blog posts

Responsive design

SEO-friendly blog structure
