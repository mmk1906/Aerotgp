Create a Personalized Dashboard

When a user logs in, the first screen should be a dashboard showing personalized information.

The dashboard should display:

• Welcome message with user name
• Profile summary
• Clubs the user has joined
• Upcoming events
• Recent announcements
• Quiz progress or scores

This gives users a quick overview of their activity on the platform.

• View quiz scores
